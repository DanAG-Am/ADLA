CSS cheatsheet and explanation

Welcome to the CSS section!

This page will talk about the usage of CSS to bring your webpage to life through color, order, shape and cohesiveness.

What is CSS?

CSS stands for Cascading Style Sheets. It is a language used to define the visual presentation of a webpage. CSS allows you to style and layout HTML elements, controlling things like colors, fonts, spacing, positioning, and responsiveness of the page. By separating the structure (HTML) from the style (CSS), it allows for easier design and maintenance of websites.

Selectors, Properties, and Values

In CSS, you target HTML elements using selectors, and you apply properties to them with corresponding values.

Selector: A CSS selector is used to target an HTML element to apply a style with a value to it. Examples of selectors include element names, classes, and IDs. IDs have the most weight as selectors, since these are the most specific. Sometimes, you may need to apply a style to a very specific element without changing its surroundings ones.
Other times, you may need to apply the same style to a large section of the page, so using less specific selectors, such as type. This enables efficiency for this case.

Property: A property defines the style to be applied. Examples of properties are color, font-size, background-color, margin, padding, etc.

Value: A value is assigned to a property. For example, for the property color, the value could be red, #ff0000, or rgb(255, 0, 0).

 
                h1 {
                color: blue; /*where property is color and its value is blue*/
                font-size: 20px; /*where the property is font size and it is applied with a value of 20 pixels*/
                }

Types of Selectors

There are different types of selectors in CSS, which allow you to target elements according to the needs and semantic style that your webpage has acquired:

  • Type Selector: It targets all instances of a specific HTML element. For example, h1 will target all h1 tags on the page.
  • Class Selector: A class selector targets elements that share the same class attribute. "class =" can be found within a tag. It is denoted by a period . before the class name. For example, .githubcontent will target all elements with the class githubcontent.
  • ID Selector: An ID selector targets a specific element with a unique ID, which can also be found within tags. It is denoted by a hash # before the ID name. For example, #mainContent will target the element with the ID mainContent.

Example of Selectors:

        /* Type Selector, targets all instances of paragraphs */
        p {
            color: green;
        }

        /* Class Selector, targets closed within githubcontent class tags */
        .githubcontent{
            background-color: yellow;
        }

        /* ID Selector, targets by a unique id or name, can also be complemented with a class instead it is repeated, but this is not a good practice. */
        #uniqueTitle {
            font-weight: bold;
        }
