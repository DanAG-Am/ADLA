Useful Terminal Commands

This page will tell you all about using commands in your terminal. Be mindful that some of these are git bash specific.

Basic commands to use on your terminal

The terminal allows you to interact with your OS.
Let's take a look at some commands that save lives.

Commands you need to know:

Command Description Example
ls Lists files and directories of the current directory (check your path) ls
cd [directory] Switches to the path of the directory specified cd /home/user/classwork
pwd Shows the directory where you currently are pwd
mkdir [name] Create a new directory. mkdir Challenge
rm [file] Deletes a specified file rm file.txt
cp [origin] [destination] Copy a file from one path to another cp file.txt /home/user/newproyect/
mv [origin] [destination] Move or rename a file or directory mv dile.txt /home/user/files/

Things you should know (from experience):

  • Be careful with the rm command, it deletes files (might sound obvious, but you might end up deleting everything...)
  • * before file names allow you to acknowledge all files with the extension. For example, if I wanted to remove all cpp files, I would type rm *.cpp
